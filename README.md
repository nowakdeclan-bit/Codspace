# School Club Management Website

A simple website for students to view school clubs, upcoming events, and sign up for clubs.

## Features
- List of clubs with descriptions
- Sign-up links (using Google Forms)
- Upcoming events calendar

## How to Use
1. Open `index.html` in your web browser.
2. View the clubs and events.
3. Click "Sign Up" links to sign up for clubs (links to Google Forms).

## Customization
- Edit the `clubs` and `events` arrays in the `<script>` tag to add your school's clubs and events.
- Replace the signup URLs with actual Google Form links.
- Modify the CSS for styling.
